## Feature description

English gloss.

This feature is also populated for `phrase` or `subphrase`, but only if they consist of just one `word` node.

## Note

See also the related feature [trans](trans.md#start) wich contains the text from the [Berean Interlinear Bible](https://berean.bible/) for this Greek word. 

## Source description

Based upon information taken from the Biblical Greek Vocabulary Builder (BGVB) developed by Oliver Glanz using the Bible OL data.
