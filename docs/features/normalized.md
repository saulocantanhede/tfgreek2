# Feature: normalized

Feature group | Feature type | Data type | Available for node types
---  | --- | --- | ---
[`Orthograpic`](featuresbygroup.md#orthograpic-features) | [`Node`](featuresbyfeaturetype.md#node-features) | `String`  | [`w`](featuresbynodetype.md#word-nodes)

## Feature description

Normalized Greek text (changed accents and diacritics to their standard forms). Also trailing punctuations are removed.

## Source description

Taken from XML attribute `normalized` of tag `w` (word).
